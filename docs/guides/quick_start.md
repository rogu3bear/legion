# Quick Start

Follow these steps to set up Legion for local development.

1. **Clone the repo.**
2. **Create a Python virtual environment.**
3. **Install dependencies.** Run `pip install -r requirements.lock`.
4. **Run checks.** Execute `scripts/ci_check.sh` to lint and test.
5. **Start the orchestrator.** Launch with `python -m legion` using the sample agent configs.

Each bullet uses short sentences. Replace configs as needed for your environment.
